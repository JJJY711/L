# 校园便民AI助手｜人工智能期末综合AIGC交互项目
## 一、项目简介
本作品为人工智能课程期末考核项目，严格按照「多模态AIGC创作 + AI辅助交互开发」要求完成。以校园生活为主题，全部素材通过AIGC工具生成，搭配零代码智能体搭建可运行人机交互应用。

## 二、多模态AIGC创作内容（全部完成四类要求）
1. **文本AIGC**
使用大模型产出选题分析、受众定位、项目简介、短视频脚本、交互页面文案，完整记录提示词迭代优化过程。
2. **图像AIGC**
AI生成校园主题海报、功能图标、食堂/图书馆场景UI素材；完成素材筛选、人工细节修改，说明无版权素材处理方案。
3. **音频AIGC**
AI生成项目解说配音、交互提示音效、轻快校园背景音乐，音画节奏与展示内容匹配。
4. **视频AIGC**
制作项目演示短视频，完整录制设计思路、图文音素材生成流程、交互智能体操作演示、成品效果。

## 三、交互作品实现说明
采用豆包智能体零代码拖拽搭建校园问答交互工具，可生成公开网页链接直接访问使用。
功能包含：课表查询、食堂窗口推荐、自习室开放咨询，输入问题即可实时得到AI回复，满足可运行、有真实交互反馈的硬性要求。
